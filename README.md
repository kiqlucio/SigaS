# SigaS - Sistema Integrado de Gestão de Assistência Social

Sistema web desenvolvido para facilitar a gestão de atendimentos e famílias em equipamentos de assistência social (como o CRAS).

## 🚀 Principais Funcionalidades

- **Dashboard:** Visão geral de atendimentos e indicadores.
- **Atendimento Individual:** Cadastro de beneficiários, prontuário eletrônico e histórico.
- **Atendimento em Grupo:** Gestão de serviços coletivos (PAIF, SCFV) e registro de encontros.
- **Administração:** Controle de usuários e manutenção do sistema.

## 🛠️ Tecnologias

- **Frontend:** HTML5, CSS3 (Vanilla) e JavaScript puro.
- **Backend:** Node.js com Express.
- **Banco de Dados:** SQLite para armazenamento local e simples.

## 📂 Estrutura do Projeto

- `assets/`: Arquivos de estilo (CSS) e lógica frontend (JS).
- `src/`: Código fonte do servidor, rotas e banco de dados.
- `docs/`: Documentação técnica e requisitos.
- `scripts/`: Scripts utilitários de banco de dados.

## ⚙️ Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor:
   ```bash
   npm start
   ```
3. Acesse no navegador: `http://localhost:3000`
